Let us now see the Anchor tag. The anchor tag defines a hyperlink, which is used to link one page to another. It is a close tag. href indicates the link's destination. Another attribute of anchor tag is the Target. There are two types of target:
- _blank (takes to some other page or opens in another tag of the browser)
- _parent (opens in the same page)
Note:- If the target is not specified the the page will open in the parent page only.

Let us now learn Meter tag. To create a meter we need to provide max, min and the value for the meter.

The next and very important tag is Table tag. The table tag is used to show some data which are related to each other. Table tag is used to create tables. The various tags used to create a complete table are as follows:
- table (to create a table)
- thead (to define the header of the table)
- tbody (to define the body of the table)
- tr (define a new row of the table)
- th (to show the heading of each column inside the table row
- td (to show the data inside the table
Note:- The other attributes of table tag are border and align. (border="X" align="Y")
